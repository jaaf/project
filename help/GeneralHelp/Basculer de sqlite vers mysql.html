Basculer d’une base sqlite3 vers une base mysql (mariadb)

Ce tutoriel existe en vidéo sur la chaîne youtube de l’application.

Installer le logiciel Python

Pour cela rendez-vous sur le site https://www.python.org/downloads/windows/, téléchargez un installateur correpondant à votre version de Windows et cliquez dessus pour l’installer

Créer un environnement virtuel et convertir la base

Pour créer un environnement virtuel tapez

C.\Users\joe> python -m venv monvenv

Activez l’environnement virtuel

C:\Users\joe> monvenv/Scripts/activate

Ensuit installez le convertisseur dans l’environnement virtuel

(monvenv) C:\Users\joe> pip install sqlite3-to-mysql

Une fois le convertisseur installé lancez la commande

(monvenv) C.\Users\joe> sqlite3mysql -f C:\Users\joe\AppData\Local\biere\mabasesqlite -d mabasemysql --mysql-password "lemotdepasse"

Dans cette commande :

  • joe est à remplacer par votre nom d’utilisateur<
  • mabasesqlite est à remplacer par le nom de votre base de données sqlite
  • mabasemysql est à remplacer par le nom de votre nouvelle base de données mysql

Créer l’utilisateur biere@localhost et lui donner un mot de passe

C:\Users\joe> mysql -u root -p lemotdepasseroot

MariaDB [(none)]> use mabasemysql;

MariaDB [(mabasemysql)]> create user biere@localhost identified by "lemotdepasseappli";

Dans ces commandes:

  • lemotdepasseroot est à remplacer par le mot de passe d’administration du serveur de base de données
  • joe est à remplacer par votre nom d’utilisateur
  • mabasemysql est à remplacer par le nom de votre base de données mysql nouvellement créée
  • lemotdepasseappli est celui qui vous sera demandé lors du démarrage de l’application après effacement des précédents choix initiaux

Effacer les précédents identifiants

Démarrez l’application et dans le menu «Divers» cliquez sur «Effacer les choix initiaux».

Redémarrez l’application est saisissez vos nouveaux choix (mysql, nom de la base, mot de passe)
