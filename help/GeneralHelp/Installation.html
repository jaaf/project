Variantes d’installation de Bière

Bière peut être installé de deux manières :

  • pour une utilisation avec le système de base de données sqlite
  • pour une utilisation avec le système de basse de données mysql

L’installation la plus simple qui est recommandée c’est la première c.-à-d. sqlite.

Sachez cependant, qu’il est possible de convertir la base de données sqlite en base de données mysql et de basculer l’application dessus.

La réciproque est également vraie

Installation pour une utilisation avec sqlite

Après avoir télchargé l’installateur sur Github, double cliquez sur cet exécutable pour lancer l’installation.

Lancez l’application à partir de l’icône sur le bureau.

L’application démarre dans un terminal (interprête de commandes).

Une première invite vous demande de choisir en sqlite et mysql

Tapez sqlite

C’est tout: l’application démarre.

Lors du prochain lancement il ne vous sera rien demandé.

Vous pouvez regarder la vidéo correpondante sur la chaîne youtube dédiée à cette application.

Installation pour une utilisation avec mysql

Vous devez d’abord installer le serveur de base de données mysql.

Création de la base de données, de l’utilisateur représentant l’application et de son mot de passe

Une fois le serveur mysql installé, ouvrez un terminal et tapez:

$ mysql -u root -p

Le caractère $ en tête de ligne symbolise l’invite de commande de l’utilisateur standard.

À la suite de cette commande, il vous sera demandé de saisir le mot de passe de l’administrateur de la base de données.

Une réponse commençant par Query OK indique que les choses se sont bien passées.

Maintenant, vous devez avoir l’invite de commande du client mysql c.-à-d. MariaDB [(none)]>

Attention, à partir de maintenant toutes les commnandes saisies derrière l’invite de commande de mysql doivent se termier par un point-virgule.

Derrière cette invite tapez la commande create database suivie du nom de la base de données

MariaDB [(none)]> create database < nom de la base de données>;

Vous pouvez choisir le nom que vous voulez (en évitant les espaces) et l’entrer sans les signes < et >

Une réponse commençant par Query OK indique que les choses se sont bien passées.

Basculez ensuite sur cette base de données avec la commande suivante:

MariaDB [(none)]> use < nom de la base de données>;

L’invite de commande devient : MariaDB [(<nom de la base>)]>

Maintenant créez l’utilisateur représentant l’application avec la commande suivante:

MariaDB [(< nom de la base de données>)]> create user biere@localhost identified by "<Mot de passe>";

en remplaçant <Mot de passe> par le mot de passe de l’utilisateur représentant l’application de votre choix.

Notez bien ce mot de pass
Vous pouvez regarder la vidéo correspondante sur la chaîne youtube dédiée à cette application.

Il ne reste plus qu’à autoriser cet utilisateur à faire ce qu’il veut sur la base de données.

MariaDB [(< nom de la base de données>)]> grant all privileges on < nom de la base de données>.* to biere@localhost;

Vous pouvez fermer le terminal maintenant.

Pour faciliter la compréhension de ce qui précède voici un exemple complet

$ mysql -u root -p

MariaDB [(none)]> create database base1;

MariaDB [(base1)]> create user biere@localhost identified by "XyZ!1234";

MariaDB [(base1)]> grant all privileges on base1.* to biere@localhost;

Installation de Bière

Après avoir télchargé l’installateur sur Github, double cliquez sur cet exécutable pour lancer l’installation.

Lancez l’application à partir de l’icône sur le bureau.

L’application démarre dans un terminal (interprête de commandes).

Une première invite vous demande de choisir en sqlite et mysql

Tapez mysql

Une nouvelle invite vous réclame le nom de la base de données

Saisissez-le

une nouvelle invite vous demande le mot de passe de l’utilisateur représentant l’application

Saisissez-le

C’est tout: l’application démarre.

Lors du prochain lancement il ne vous sera rien demandé.
